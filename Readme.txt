This simple app can be used as following:
1. Select the data source from the drop-down list and type in the symbol.
2. The default date range is given, while you can change it with the calendar
3. In order to see the price time series of the data, click on the first tab on the right side. Notice that if the symbol can not be found in the data source, you will get error message, which indicates that you need to check if the symbol and/or data source are correct. Sample symbols are the following:
SPY(SP500),GOOG for yahoo, XAU/XAG or  USD/JPY for OANDA 
4. Click on the second tab to see the calculated return time series, which is same time range as the price time series
5. Click on the last tab to check out the exploratory analysis result of the return time series.
